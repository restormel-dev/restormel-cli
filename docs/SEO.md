## README (both repos)

- **First line:** Include "Restormel", "Next.js", and "CLI" or "Starter".
- **First paragraph:** 1–2 sentences with phrases: Next.js, security, scaffold, template.
- **Cross-link:** Each README links to the other repo (See also / Related).
